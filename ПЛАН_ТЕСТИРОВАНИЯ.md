# 🔧 План тестирования двух парсеров

## ❗ Важно: отключите VPN перед тестированием!

## 📋 Чек-лист для проверки

### 1. **Парсер ЕИС** 
- [ ] **Ссылки корректные**: не ведут на ошибки 404
- [ ] **Названия правильные**: НЕ статусы типа "Подача заявок..."  
- [ ] **RSS работает**: получает данные без ошибок
- [ ] **ID извлекаются**: корректные regNumber из URL
- [ ] **Фильтры работают**: находит тендеры по ключевым словам

### 2. **Парсер Портала поставщиков**
- [ ] **Селениум подключается**: к https://zakupki.mos.ru
- [ ] **Ссылки НЕ на подписи**: исключены /sign, /certificate  
- [ ] **Названия извлекаются**: реальные названия котировок
- [ ] **ID корректные**: из параметров или пути URL
- [ ] **Фильтры работают**: поиск по ключевым словам

### 3. **Общие проверки**
- [ ] **Дедупликация**: не отправляет повторно одни тендеры
- [ ] **Telegram уведомления**: корректно отправляются  
- [ ] **Логирование DEBUG**: подробные логи процесса
- [ ] **База данных**: корректно сохраняет тендеры
- [ ] **Обработка ошибок**: не падает при сбоях

## 🚀 Команды для тестирования

### Быстрая проверка синтаксиса:
```cmd
python -m py_compile parsers/eis_parser.py
python -m py_compile parsers/suppliers_portal_parser.py
python -m py_compile main.py
```

### Полный запуск:
```cmd
запуск_парсера.bat
```
ИЛИ
```cmd
.\.venv\Scripts\python.exe main.py
```

### Проверка логов:
```cmd
tail -f parser.log
```

## 🔍 На что обратить внимание в логах

### ✅ **Хорошие сигналы:**
```
RSS: исходная ссылка: ...
RSS: исправленная ссылка: https://zakupki.gov.ru/...
Извлечен regNumber: 0373200024925000001
Найден тендер: "Поставка наградной продукции..."
Фильтр ПРИНЯТ: найдено ключевое слово 'грамота'
```

### ❌ **Плохие сигналы:**
```
Не найден элемент с ссылкой на тендер
Пустое или некорректное название тендера  
Некорректный URL: ...
Таймаут для https://zakupki.gov.ru/...
```

## 📊 Ожидаемые результаты

### После успешного запуска:
1. **В логах** увидите процесс парсинга обеих площадок
2. **В Telegram** придут уведомления с рабочими ссылками  
3. **В базе tenders.db** сохранятся найденные тендеры
4. **Без ошибок** в логах (кроме нормальных предупреждений)

## 🛠 Если что-то не работает

### Проблема с ЕИС:
- Проверьте VPN (должен быть отключен)
- Проверьте интернет соединение
- Посмотрите лог на ошибки timeout

### Проблема с Порталом поставщиков:  
- Проверьте установку Chrome/Edge
- Убедитесь что Selenium драйвер доступен
- Проверьте доступность zakupki.mos.ru

### Общие проблемы:
- Убедитесь что активирована виртуальная среда `.venv`
- Проверьте файл config.py на корректность
- Посмотрите детальные DEBUG логи

## 🎯 Критерии готовности

**Парсеры готовы к продуктивной работе когда:**

✅ Оба парсера работают без критических ошибок  
✅ Ссылки на тендеры открываются и корректны  
✅ Названия тендеров извлекаются правильно (не статусы)  
✅ Фильтрация по ключевым словам работает  
✅ Telegram уведомления приходят с корректной информацией  
✅ Дедупликация предотвращает повторные уведомления
