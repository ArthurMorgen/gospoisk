## 🔧 Если нужно настроить

### Включить портал поставщиков:
В файле `config.py` измените:
```python
'suppliers_portal': {
    'enabled': True,  # Включить
```

### Изменить ключевые слова:
В файле `config.py` в разделе `FILTER_CONFIG['keywords']`
