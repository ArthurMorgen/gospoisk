# 🤖 Telegram Бот для Парсинга Госзакупок

## 📋 Что это?

Telegram бот с **кнопками** для получения тендеров **по запросу**.

Вы сами выбираете когда и с какой площадки получить тендеры!

---

## 🎯 Возможности

### Кнопки в боте:

1. **📋 ЕИС** - Актуальные тендеры с zakupki.gov.ru
2. **🏛️ Портал поставщиков** - Котировочные сессии с zakupki.mos.ru
3. **📊 Статистика** - Статистика по базе данных

### Фичи:

- ✅ **Листание результатов** - по 5 тендеров на страницу с кнопками ◀️ ▶️
- ✅ **Только активные** - показываются только актуальные котировки
- ✅ **Кнопка "Назад"** - всегда можно вернуться в главное меню
- ✅ **Команда /start** - быстро вернуться к началу

---

## 🚀 Запуск

Просто запустите:
```cmd
запуск.bat
```

Откроется окно с ботом. Откройте Telegram и напишите `/start` вашему боту!
```cmd
запуск_бота.bat
```

## 🔄 Два режима работы

### Режим 1: Автоматический (старый)
- Всегда работает в фоне
- Присылает каждый новый тендер
- Нельзя пропустить

**Плюсы:** Не пропустишь ни одного тендера
**Минусы:** Много уведомлений

### Режим 2: По запросу (новый)
- Работает только когда вы нажимаете кнопку
- Показывает актуальные тендеры
- Контролируете поток

**Плюсы:** Контроль, меньше спама
**Минусы:** Нужно самому запрашивать

## 📝 Что дальше?

### Для новых площадок:

Когда будете добавлять новые сайты, просто:

1. Создайте парсер в `parsers/`
2. Добавьте кнопку в `bot_interactive.py`:
```python
InlineKeyboardButton("🏢 Новая площадка", callback_data='parse_new')
```
3. Добавьте обработчик:
```python
elif query.data == 'parse_new':
    await parse_new_platform(query)
```

**Готово!** Новая площадка доступна через кнопку!

## 💰 Преимущества подхода с кнопками

✅ **Масштабируемость** - Каждый новый сайт = новая кнопка
