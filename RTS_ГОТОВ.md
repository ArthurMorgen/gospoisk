## 🚀 КАК ДОБАВИТЬ НОВУЮ ПЛОЩАДКУ:

Теперь это просто! Следуйте шаблону RTS-Tender:

### Шаг 1: Создать парсер

```python
# parsers/new_platform_parser.py
from parsers.rts_tender_parser import RTSTenderParser

class NewPlatformParser(RTSTenderParser):
    # Скопировать структуру
    # Изменить селекторы
```

### Шаг 2: Добавить конфиг

```python
# config.py
'new_platform': {
    'enabled': True,
    'name': 'Название',
    'base_url': 'https://...',
    'search_url': 'https://...',
    'use_selenium': True  # Если нужен
}
```

### Шаг 3: Добавить кнопку в бот

```python
# bot_interactive.py
InlineKeyboardButton("🆕 Название", callback_data='parse_new')
```

**ГОТОВО!** 🎉
