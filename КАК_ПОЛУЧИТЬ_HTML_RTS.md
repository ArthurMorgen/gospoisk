## ✅ РЕШЕНИЕ 2: НАЙТИ API ENDPOINT (БЫСТРЕЕ!)

### Шаг 1: Открыть DevTools
- Откройте https://www.rts-tender.ru/poisk/poisk-44-fz/
- Нажмите `F12` (или `Ctrl+Shift+I`)

### Шаг 2: Открыть вкладку Network
- Перейдите на вкладку **Network** (Сеть)
- Установите фильтр **XHR** или **Fetch**

### Шаг 3: Обновить страницу
- Нажмите `Ctrl+R` или кнопку "Обновить"
- Или введите поисковый запрос

### Шаг 4: Найти запрос с тендерами
Ищите запросы типа:
- `api/search`
- `api/tenders`
- `search?query=...`
- `getTenders`
- Любые JSON/XML ответы

### Шаг 5: Скопировать данные

**Для каждого найденного API запроса:**

1. **URL запроса:**
   - Правый клик на запрос → Copy → Copy URL
   - Пример: `https://www.rts-tender.ru/api/search?q=грамота&page=1`

2. **Параметры (Query String Parameters):**
   - Откройте запрос
   - Вкладка **Headers** → секция **Query String Parameters**
   - Скопируйте все параметры

3. **Пример ответа (Response):**
   - Вкладка **Response** или **Preview**
   - Скопируйте JSON/XML (первые 100-200 строк достаточно)

4. **Заголовки (Headers):**
   - Вкладка **Headers** → секция **Request Headers**
   - Скопируйте важные заголовки (Authorization, Cookie, X-API-Key)

### Шаг 6: Отправить мне

Создайте файл `rts_api_info.txt` с этой информацией:
```
=== URL ЗАПРОСА ===
https://www.rts-tender.ru/api/search?q=грамота&page=1

=== ПАРАМЕТРЫ ===
q: грамота
page: 1
limit: 20

=== ЗАГОЛОВКИ ===
Authorization: Bearer ...
Cookie: session=...

=== ПРИМЕР ОТВЕТА (JSON) ===
{
  "items": [
    {
      "id": 12345,
      "title": "Закупка грамот",
      "price": 50000,
      ...
    }
  ],
  "total": 100
}
```
