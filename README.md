# 🤖 Парсер Госзакупок

Автоматический мониторинг государственных закупок с уведомлениями в Telegram.

## 🚀 Быстрый запуск

Просто запустите:
```cmd
запуск.bat
```

Выберите нужный режим из меню!

## Возможности

- 🔍 Автоматический поиск новых тендеров на ЕИС и Портале поставщиков
- 🎯 Фильтрация по ключевым словам, категориям и ценовому диапазону
- 📧 Уведомления по email и Telegram
- 💾 Сохранение данных в локальной базе SQLite
- 📊 Статистика и отчеты
- ⏰ Настраиваемый интервал проверки

## Установка

1. Клонируйте репозиторий или скачайте файлы
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию в файле `config.py`

## Настройка

### Основные параметры

В файле `config.py` настройте:

- `PARSING_CONFIG['check_interval']` - интервал проверки в секундах (по умолчанию 300 = 5 минут)
- `FILTER_CONFIG['keywords']` - список ключевых слов для поиска
- `FILTER_CONFIG['exclude_keywords']` - список исключающих слов

### Настройка уведомлений

#### Email уведомления:
```python
NOTIFICATION_CONFIG['email'] = {
    'enabled': True,
    'smtp_server': 'smtp.gmail.com',
    'smtp_port': 587,
    'username': 'your_email@gmail.com',
    'password': 'your_app_password',
    'recipients': ['recipient@example.com']
}
```

#### Telegram уведомления:
```python
NOTIFICATION_CONFIG['telegram'] = {
    'enabled': True,
    'bot_token': 'YOUR_BOT_TOKEN',
    'chat_ids': ['YOUR_CHAT_ID']
}
```

### Настройка фильтров

```python
FILTER_CONFIG = {
    'keywords': ['программное обеспечение', 'разработка', 'IT'],
    'exclude_keywords': ['ремонт', 'строительство'],
    'min_price': 100000,
    'max_price': 5000000,
    'regions': ['Москва', 'Московская область'],
    'categories': ['Информационные технологии']
}
```

## Логирование

Все события записываются в файл `parser.log` и выводятся в консоль. Уровень логирования настраивается в `config.py`.

## База данных

Используется SQLite база данных `tenders.db` со следующими таблицами:
- `tenders` - информация о тендерах
- `categories` - категории тендеров
- `tender_categories` - связи тендеров и категорий
- `notifications` - журнал уведомлений

## Безопасность

- Не храните пароли и токены в открытом виде
- Используйте переменные окружения для чувствительных данных
- Регулярно обновляйте зависимости

## Возможные проблемы

1. **Блокировка по IP** - добавьте задержки между запросами
2. **Изменение структуры сайтов** - обновите селекторы в парсерах
3. **Ошибки уведомлений** - проверьте настройки email/Telegram

## Разработка

Для добавления новой площадки:
1. Создайте новый парсер, наследующий от `BaseParser`
2. Реализуйте методы `search_tenders()` и `parse_tender_details()`
3. Добавьте конфигурацию площадки в `config.py`
4. Зарегистрируйте парсер в `main.py`
